# resume/

Place your resume file here. Supported formats:

- `.pdf` — recommended
- `.docx` — Word documents
- `.txt` — plain text

## Rules

- **Any filename is accepted** — the system always uses the newest file.
- If you replace your resume, delete the old file or save with a newer timestamp.
- Do not hardcode filenames anywhere in the project.

## Example

```
resume/
  Sravya_Resume_2026.pdf   ← this will be auto-detected
```

> This directory is tracked in git but resume files are listed in `.gitignore`.
> Never commit your actual resume to a public repository.
